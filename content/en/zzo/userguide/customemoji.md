You can set the custom emoji icons in your `params.toml` file.

```params.toml
...
writtenTimeIcon = "📅"
modifiedTimeIcon = "📝"
readingTimeIcon = "☕"
authorIcon = "✍️"
pagePvIcon = "👀"
tagIcon = "🏷️"
publicationIcon = "📚"
typeIcon = "🎯"
```
